Simple Web Server Commands Guide

You can test the server with the following curl commands:

Upload a file:

curl -X POST --data-binary @myfile.wav http://localhost:7878/files

List all files:

curl http://localhost:7878/files

List files with filter:

curl "http://localhost:7878/files?wav"

Get file content:

curl http://localhost:7878/content/filename.wav > downloaded_file.wav

Get file metadata:

curl http://localhost:7878/metadata/filename.wav

Get all WAV files with a maximum duration of 300 seconds:

curl "http://localhost:7878/files?maxduration=300"

Get all WAV files with "tone" in the filename and a maximum duration of 180 seconds:

curl "http://localhost:7878/files?filter=tone&maxduration=180"

Additional Commands:

curl "http://localhost:7878/files?filter=wav"
