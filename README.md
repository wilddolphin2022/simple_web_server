# Simple Web Server

Learn Rust with this example

## Using

To build, [install Rust], and then:

```bash
git clone https://github.com/wilddolphin2022/simple_web_server
cd simple_web_server
cargo build
cargo test
cargo run
```

[install Rust]: https://www.rust-lang.org/tools/install
